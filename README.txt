RIENTRI MANUTENTIVI

1. Copia tutta la cartella in:
   C:\Users\Marco Failla\Desktop\Rientri manutentivi

2. Se vuoi, sostituisci il file regionale.png con la tua immagine definitiva.

3. Esegui:
   - installa_requisiti.bat
   - avvia_app.bat

L'app:
- usa asset DOR.xlsx per DR / IMC / ROTABILE
- aggiorna direttamente Rientrimanutentivi.xlsx
- aggiunge automaticamente le colonne di servizio: ID, STATO, DATA CREAZIONE, ULTIMO AGGIORNAMENTO
- invia email tramite Outlook desktop; se Outlook non parte, crea una bozza cliccabile
